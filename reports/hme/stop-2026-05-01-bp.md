# STOP — eksport PSD z canvas renderu (BP)

## Zakres

- **`ag-psd`** przeniesione do **dependencies** (chunk lazy-load ~240 kB raw / ~77 kB gzip).
- **`src/engine/filmLabExportPsdFromCanvas.js`** — `encodeFilmLabExportPsdFromCanvas(canvas, { layerName })` → bytes PSD (`application/vnd.adobe.photoshop`).
- **`useFilmLabEngine.exportImage`** — gdy `fileFormat === 'psd'`: ostrzenie jak raster → PSD zamiast `encodeFilmLabExportCanvas`; manifest `export.fileFormat: psd`; sidecary **before/mask recipe** używają **`rasterFf`** (`jpeg` przy primary PSD).
- **`normalizeFilmLabExportModalFileFormat`** w `filmLabExportFormats.js`; **`FilmLabExportModal`** zachowuje `psd` w prefs (bez pillu w UI).
- **`film-lab-export-perf-gates.mjs`** — asercje PSD / modal normalizer.

## Testy

- `npm run lint`, `npm run test:film-lab-export-gates`, `npm run build` — PASS.

## Uwaga produktowa

- PSD jest **eksperymentalny** (prefs `fileFormat: psd`); pełny wybór w UI — osobny krok.
